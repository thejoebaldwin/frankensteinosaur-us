MySQL Database and File Upload Interface in Visual Basic

This application was developed to allow the YMAA Taijiquan of Wisconsin Director to update the news and event webpages at www.ymaa-wisconsin.com (which are populated by the database) without requiring any knowledge of MySQL syntax. There is a file upload function for ease in transferring files to the webspace as well. The application was developed using Microsoft Visual Studio 6.0
